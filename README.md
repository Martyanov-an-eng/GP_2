# GP_2

Мы сотрудники хэдж-фонда, хотим понять как сегодня настроен рынок, то есть анализировать его сентемент, для этого мы берем новости из newsapi и прогоняем его через vaderSentiment.
Это сделано в ноутбуке NewsApi. Результаты этого исследования можно дальше будет применять для анализа того, как изменяются цены акции в зависимости от настроения рынка.
